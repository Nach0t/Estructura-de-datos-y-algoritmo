//
Escriba un algoritmo en pseudocódigo, que permita sumar todos los elementos de un vector de n números enteros, tanto como caso no recursivo (30%) y como caso recursivo (20%).
//

**Método No Recursivo:

1. Inicializar con una variable suma a0.
2. Para cacualquier elemento de n en el vector:
    2.1. USas el valor n para sumarlo a la variable suma.
3. retorna el valor de suma.


**Método Recursivo:

1. Definir una función sumaRecursiva que acepte dos parámetros: vector y índice.
2. Si el índice es igual a la longitud del vector (condición base), devolver 0. 
Esto significa que hemos terminado de recorrer el vector.
3. Si el índice es menor que la longitud del vector, devolver el elemento en el 
índice actual más el resultado de llamar a sumaRecursiva con el vector y el índice 
incrementado 
en 1.
